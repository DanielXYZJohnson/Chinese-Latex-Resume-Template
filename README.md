# LaTeX 中文简历模板

这是一个简洁的中文简历LaTeX模板，适配**在线LaTeX编辑器（如LoogTex）**，支持快速编译与PDF生成。
模板包含基础个人信息、教育背景、科研经历等模块，开箱即用。

---

## 📂 文件结构

```
Resume_Template/
├── Resume_template.tex     # 主LaTeX模板文件（在此编写简历内容）
└── images/
    └── zhangwei.png        # 示例头像图片（需替换为你的照片）
```
